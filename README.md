# UGC Ad Pack — Cafe Marketing Prompt System

This repository contains a reusable **UGC (User Generated Content) ad creation system** designed for promoting a **local cafe business** using short-form video platforms such as:

- Instagram Reels
- YouTube Shorts

The system helps generate **authentic UGC-style ads** that feel like a real customer experience rather than a traditional advertisement.

Instead of polished marketing videos, these scripts mimic **real customer recommendations**, which perform significantly better on social media platforms.
